# Ref Movements

A Python app that uses MediaPipe to detect basketball ref movements such as travel and double dribble

## Run App

## How to use
Download the code and start the virtual environment 

```bash
    #Windows
    venv\Scripts\activate
    #Mac
    source .venv/bin/activate
```
Windows

```bash
    python -m pip install -r requirements.txt
```
Mac

```bash
  pip3 install -r requirements.txt
```
